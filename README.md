# Flutter Animation Project

A Flutter project focused on learning and implementing different types of animations in Flutter.

## Features

* Implicit and explicit animations.
* Animated widgets using Flutter’s built-in animation system.
* Smooth transitions between UI states.
* Custom animations with AnimationController.
* Clean and reusable UI components.

## Tech Stack

* Flutter
* Dart
* Flutter Animation Framework

## Getting Started

### Prerequisites

* Flutter SDK installed
* Dart SDK
* Android Studio or VS Code

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Mina8496/animation.git
```

2. Navigate to the project directory:

```bash
cd animation
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```

## Learning Objectives

This project was created to:

* Understand Flutter animation fundamentals.
* Work with AnimationController and Tween.
* Build smooth and interactive UI experiences.
* Practice state-driven UI updates with animations.
